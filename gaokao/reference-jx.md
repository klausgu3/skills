## 三、整校排除（用户要求「不要师范」时）

校名含：**师范大学、师范学院、科技师范大学** → 整校不报。

示例已剔除：赣南师大、江西师大、湖二师、汉江师院、南昌师范学院、江西科技师范大学…

## 七、组内专业默认顺序（物化地 / 物理类）

```
电信 → 自动化/机器人 → 计算机/软件 → 机设/智能制造 → 电气(末位)
```

禁填：师范、医学、法学、金融、市场营销、化学/生物类（除非用户要）。
